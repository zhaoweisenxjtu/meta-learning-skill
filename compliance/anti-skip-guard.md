# 反偷懒守护协议

## 触发条件

以下任一情况发生时，agent 必须立即停止当前教学，执行守护协议：

### 触发信号
1. 用户说"直接讲重点""跳过基础""我大概知道"
2. agent 发现自己跳过了直觉构建直接给定义
3. agent 发现自己没有预警迷思概念
4. agent 发现自己没有做即时检验
5. agent 一次性讲了多个概念但中间没有穿插检验题
6. 教学回复中缺少合规评分卡
7. 连续 2 次合规率 < 100%

### 守护协议

当触发时，agent 必须：

1. **立即停止**当前教学流程
2. **输出守护标记**：`🛡️ 反偷懒守护激活：检测到[触发原因]`
3. **回补缺失步骤**：从第一个被跳过的步骤重新执行
4. **输出完整合规评分卡**

## 强制规则

- **禁止**在直觉构建完成前给出形式化定义
- **禁止**在迷思预警完成前进入例题讲解
- **禁止**在即时检验完成前进入下一个知识点
- **禁止**一次性讲完多个概念而不穿插检验题
- **禁止**在任何教学回复中省略合规评分卡
- **禁止**在教学结束后不调用引擎记录命令
