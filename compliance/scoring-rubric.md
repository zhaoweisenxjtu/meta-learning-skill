# 教学合规评分细则

## 评分维度（22 项）

| 维度 | 0分（未执行） | 1分（部分执行） | 2分（完全执行） |
|------|------------|--------------|--------------|
| 模式判定 | 未判定 | 判定但无依据 | 判定+输出依据 |
| 用户与路线检索 | 未检索 | 检索不完整 | 完整检索+用户确认 |
| 概念自评 | 未做 | 只列概念未让用户解释 | 列概念+用户解释+判定 |
| 快速诊断测试 | 未出题 | 出题但未对比自评 | 出题+评分+偏差判定 |
| 偏差判定 | 未判定 | 判定但未处理 | 判定+针对性处理 |
| 教学准备 | 未准备 | 部分检查 | 缓存+历史+迷思全检 |
| 直觉构建 | 直接给定义 | 有类比但不直观 | 类比+可视化+日常经验 |
| 动机说明 | 未说明 | 一句话带过 | 完整回答"为什么需要" |
| 概念讲解 | 未给出定义 | 不精确 | 精确定义+公式+推导 |
| 迷思预警 | 未预警 | 预警但未纠正 | 预警+纠正+原因 |
| 边界条件 | 未说明 | 模糊说明 | 明确适用范围+失效场景 |
| 知识连接 | 未连接 | 单一连接 | 来源+关联+去向 |
| 即时检验 | 未检验 | 检验但无信心校准 | 检验+信心校准+构造反例 |
| 分类任务 | 未做 | 做了但未评分 | 完成+正确率 |
| 概念图 | 未做 | 做了但不完整 | 完成+完整性评估 |
| 费曼讲解 | 未做 | 做了但未评估 | 完成+L5检验 |
| 引擎记录·互动 | 未调用 | 调用但参数不全 | 完整调用 |
| 引擎记录·SM-2 | 未调用 | 调用但评分不准 | 完整调用 |
| 引擎记录·质量 | 未调用 | 调用但未评估 | 完整调用 |
| 引擎记录·迷思 | 未调用 | 有迷思未记录 | 完整记录 |
| 引擎记录·薄弱 | 未调用 | 调用但未分析 | 完整分析 |
| 教学结束自检 | 未自检 | 部分自检 | 逐条确认 |

## 合规等级

- 🟢 优秀：38-44 分（22 项全部接近 2 分）
- 🟡 合格：30-37 分
- 🟠 待改进：22-29 分
- 🔴 不合规：<22 分 → 触发反偷懒守护协议
